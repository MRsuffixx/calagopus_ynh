Calagopus is a modern, open-source game server management panel built with Rust and React. It provides a fast, secure interface for deploying, monitoring, and maintaining game servers — designed for everyone from solo homelabbers to large hosting operators.

It draws inspiration from Pterodactyl but is written from scratch in Rust, delivering dramatically better performance alongside memory-safety guarantees. The panel ships with a live console, file manager, backup browser, schedule tasks, subuser management, WebAuthn authentication, and a comprehensive REST API.
